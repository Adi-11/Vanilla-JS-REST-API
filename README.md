# Vanilla-JS-REST-API
> Created a REST API using Node.js only, without Express

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```
# Install dependencies
> npm install

# Run in develpment
> npm run dev

# Run in production
> npm start
